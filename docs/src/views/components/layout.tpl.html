.flex--0

Use .flex--0 for sections that shouldn't grow and shink. It's very similar to .minor but without the "display: flex" declaration.

.flex--1

Use .flex--1 for sections that should grow and shink. It's very similar to .major but without the "display: flex" declaration.

.flex--0

Use .major or .minor classes only for sections that contains another layout inside.
The .flex--1 or .flex--0 should be used for sections that contains content itself like paragraphs, lists, tables etc.
There are many cases where sections may have a "flex" display mode (toolbar needs it for easy icons centering), for that you can use the .minor class but in terms of readability it's prefferable to use two classes - .flex and .flex--0.
