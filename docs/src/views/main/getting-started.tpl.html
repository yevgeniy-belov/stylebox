Getting started
Get Stylebox

Getting Stylebox is simple - just clone the Stylebox GIT repository.

Full Access

To get full access to the repository you need to login with your.name@hp.com and your PC password.

What's included

Grid system

Stylebox is using the grid system taken "as is" from the Twitter Bootstrap.

Layout system

The layout system used in Stylebox is based on the Flexible Box module that makes layouting fun again.

Icons font

Stylebox is using vector icons made by HP Software designers and generated to a font using grunt-webfont. The list of icons is constantly growing.

Building Blocks

Building Blocks is a modular CSS micro-framework that helps designing in the browser.
It contains the smallest possible pieces of UI components like borders, backgrounds, margins and paddings.

UI Components

A bunch of well designed, pixel perfect UI components.

Naming convention

Stylebox is using the BEM naming convention.
BEM is an abbreviation of three entities: a Block, an Element and a Modifier.

BEM

BEM is a methodology of web projects development, that allows you to divide an interface into logically independent blocks.

Browser support

Many of the Stylebox componets are based on the Flexible Box technology which means that you need at least IE10 to run it correctly.

Can I Use

If you want to see the complete list of supported browsers you can check it at caniuse.com.
